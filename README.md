# HR-Analytics-Dashboard
An interactive HR Analytics Dashboard built using Power BI and SQL to analyze employee attrition, workforce demographics, and job satisfaction trends. This project helps HR teams make data-driven decisions by identifying key factors affecting employee turnover.
## 📊 Dashboard Overview
This dashboard provides insights into:
##### 👥 Total Employees
##### 📉 Attrition Rate
##### 💼 Active Employees
##### 🎂 Employee Age Analysis
##### 🏢 Department-wise Attrition
##### 🎓 Education Field Attrition
##### ⭐ Job Satisfaction Rating
##### 👨‍💼 Attrition by Job Role
## 🛠️ Tools & Technologies Used
##### Power BI
##### SQL
##### Data Cleaning & Transformation
##### DAX Measures
##### Interactive Visualizations
